Em que consiste o login?

A funcionalidade de login deve permitir ao usuário humano digitar suas credenciais (username e senha), deve enviá-las para algum endpoint no servidor para que sejam verificadas e deve baixar e armazenar uma credencial de acesso (um token) para outras interações com o servidor.
