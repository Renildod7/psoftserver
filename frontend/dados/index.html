DADOS

Aqui vamos implementar uma área de acesso controlado.

Como funcionará?

Esta página deve buscar dados em algum endpoint servidor. Contudo, por ser de acesso controlado, esse endpoint deve exigir que o usuário tenha feito login. Para isso, o endpoint exige o uso de credenciais na forma de um token de acesso (que deve ter sido obtido na tela de login).

Se o usuário não estiver logado, esta página deve exibir uma mensagem e um link para o login.
